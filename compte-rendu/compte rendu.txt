j'ai documenté  aujourd hui sur html "hyper text markup language" est un Langage de balisage utilisé pour la création de pages web,

les versions du html html html 2.0 html 3.2 Xhtml html 5.
types de balisage j'ai trouvé huit balise et ils sont : 
 	*balise de formailaire : <form> , <input> , <label>
	*balise de tableaux : <table>,<tr>,<td>,<th>
	*balise de liste : <ul> <li>
	*balise de 1er niveau : head && body && html => head && body.
	*balise en tete : <meta>,<title>,<link>
	*balise de structeration du text : <strong>,<h1>---<h6>,<p>...
	*balise Sectionnante : <header>,<footer>,<section>,<article>
	*balise generique : <div>,<span>
